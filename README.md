# logicstar-cli

Install the [logicstar](https://logicstar.ai) CLI on your machine.

## Install

```sh
npx github:logic-star-ai/logicstar-cli-releases
```

Installs to `~/.local/bin/logicstar`. Make sure that directory is on your `PATH`, then run `logicstar`.

Requires Node.js 18+.

## Updates

`logicstar` checks for updates in the background. To check now:

```sh
logicstar update
```

## Supported platforms

macOS (Apple Silicon, Intel) · Linux (x86_64, ARM64).
